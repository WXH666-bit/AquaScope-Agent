# 启动命令与模块调用手册

## 1. 推荐启动

```cmd
cd /d F:\rag\AquaBio-AgentRAG
start_chat_assistant.cmd
```

该命令依次启动：

1. RAG-Anything MCP：`http://127.0.0.1:8765/mcp`
2. FastAPI：`http://127.0.0.1:8000/docs`
3. Streamlit：`http://127.0.0.1:8510`
4. MCP warmup
5. LangGraph、BGE-M3、Chroma warmup

当前版本已经做了容错：如果 RAG-Anything MCP 启动或 warmup 超时，
FastAPI 和 Streamlit 仍然会继续运行；这时图谱检索会在状态中显示不可用，
普通 Chroma 检索、图片上传和基础图文问答不会被 MCP 卡死。

显示三个后台窗口：

```cmd
start_chat_assistant.cmd
```

隐藏窗口启动：

```powershell
powershell -ExecutionPolicy Bypass -File scripts\start_chat_assistant.ps1
```

停止：

```cmd
stop_chat_assistant.cmd
```

注意：停止脚本现在只会停止命令行中明确包含 `AquaBio-AgentRAG`、
`api_app:app`、`mrag_app.py` 或 `aquabio_raganything.mcp_server`
的进程，避免旧 PID 被系统复用后误关 QQ、浏览器等其他程序。

分布图验证示例：

```text
第一轮：鲨鱼是什么生物？
第二轮：给我它的分布图
```

系统优先返回 PDF 图片实体库中角色为 `distribution_map` 的图片；本地没有
对应地图时才从 Wikimedia Commons 获取。普通生物实例图不会冒充分布图。

## 2. 分别启动

```cmd
set PYTHONPATH=F:\rag\AquaBio-AgentRAG\src

.\.venv-raganything\Scripts\python.exe -m aquabio_raganything.mcp_server
.\.venv\Scripts\python.exe -m uvicorn api_app:app --host 127.0.0.1 --port 8000
.\.venv\Scripts\python.exe -m streamlit run mrag_app.py --server.port 8510
```

MCP 使用 HTTP 时：

```cmd
set AQUABIO_RAG_MCP_TRANSPORT=streamable-http
set AQUABIO_RAG_MCP_PORT=8765
```

## 3. 主要模块调用顺序

### 3.1 前端到回答

```text
mrag_app.py
  → POST /api/chat/tasks
  → ChatService.submit_chat
  → ChatService._run_chat_task
  → ChatService.chat
  → AquaBioMRAGWorkflow.invoke
  → LangGraph
  → Qwen + RetrievalAgent + MCP
  → evidence / trace / images / answer
  → WebStore.save_turn
  → Streamlit 渲染
```

### 3.2 图片输入

```text
上传图片
  → /api/uploads/image
  → 保存 data/mrag/uploads/
  → vision_node 调用 Qwen VLM
  → caption + visual_features + candidates
  → 本地图文向量 + PDF 图片 MCP
  → 证据融合
  → Qwen 生成带引用回答
```

前端操作要点：

1. 在“图片附件”选择图片后，系统会自动上传并加入“本轮附件”。
2. 如果没有看到“本轮附件”区域里的缩略图，说明本轮问题不会携带图片。
3. 如果只问“这是什么生物”但后端没有收到图片，系统会立即提示补图，
   不再进入长时间 RAG/MCP 检索。
4. 如果图片已收到但关闭了“图像理解”，系统会明确提示图片已收到、
   但视觉分析未启用，不会再误说“没有上传图片”。

### 3.3 PDF 知识库查询

```text
用户问题
  → pdf_retriever
  → RetrievalAgent.search_pdf
  ├─ Chroma dense retrieval
  ├─ BM25 lexical retrieval
  └─ RAG-Anything MCP hybrid/graph retrieval
  → weighted RRF
  → context builder
  → Qwen answer
```

### 3.4 PDF 图片查询

```text
“印刷页413的一张生物图”
  → requested_page 解析页码
  → raganything_image_search MCP
  → pdf_image_captions.jsonl
  → 图片角色筛选
  → 最终只保留一张最佳 PDF 原图
```
